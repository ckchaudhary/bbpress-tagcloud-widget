Display bbPress tag cloud in a widget.

== Description ==

You can display the bbPress tag cloud using a shortcode. This plugin adds a bit, by wrapping the shortcode in a widget, which you can use anywhere by just drag-and-drop.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
